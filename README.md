# Bezemdagdelen_2025_2026

Op deze pagina vind je de plaatsen en tijden van de ingeplande bezem-dagdelen voor het schooljaar 2025_2026.
In de regel vallen die dagdelen op de donderdagmiddag. Bij uitzondering worden dagdelen verzet naar een ander moment. Het is geen vast lokaal. Het blijkt dat studenten die actief met de vakken bezig zijn en de bezemdagdelen bezoeken, een grote kans hebben om die vakken te kunnen aftikken.

De bezemdagdelen zijn in de eerste plaats bedoeld voor ondersteuning van de vakken:
ALDS, Vision, MRB, OOPC, DIT, IPASS.

Het valt niet direct onder mijn toko, maar ik kan ook meekijken/meedenken met:
CPSE2 en EOS
